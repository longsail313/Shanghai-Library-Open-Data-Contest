HTML列表教学

1. 无序列表（Unordered List）

无序列表使用 <ul> 标签，每个列表项使用 <li> 标签：

  • 苹果
  • 香蕉
  • 橙子
<ul>
    <li>苹果</li>
    <li>香蕉</li>
    <li>橙子</li>
</ul>

2. 有序列表（Ordered List）

有序列表使用 <ol> 标签，每个列表项使用 <li> 标签：

  1. 第一步
  2. 第二步
  3. 第三步
<ol>
    <li>第一步</li>
    <li>第二步</li>
    <li>第三步</li>
</ol>

3. 定义列表（Definition List）

定义列表使用 <dl> 标签，包含 <dt>（定义术语）和 <dd>（定义描述）：

HTML
超文本标记语言，用于创建网页
CSS
层叠样式表，用于设计网页样式
<dl>
    <dt>HTML</dt>
    <dd>超文本标记语言，用于创建网页</dd>
    <dt>CSS</dt>
    <dd>层叠样式表，用于设计网页样式</dd>
</dl>

4. 嵌套列表

列表可以嵌套使用，创建多级列表：

  • 水果
    • 苹果
    • 香蕉
  • 蔬菜
    • 胡萝卜
    • 白菜
<ul>
    <li>水果
        <ul>
            <li>苹果</li>
            <li>香蕉</li>
        </ul>
    </li>
    <li>蔬菜
        <ul>
            <li>胡萝卜</li>
            <li>白菜</li>
        </ul>
    </li>
</ul>

  2. 使用大写字母
  1. 使用小写字母
  2. 使用小写字母
  1. 使用大写罗马数字
  2. 使用大写罗马数字
  1. 使用小写罗马数字
  2. 使用小写罗马数字

6. 列表使用注意事项

  • 列表项（<li>）必须包含在列表容器（<ul>、<ol>、<dl>）中
  • 定义列表中的 <dt> 和 <dd> 必须成对出现
  • 列表可以嵌套，但要注意层级不要过深
  • 列表项可以包含其他HTML元素
